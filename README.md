# ⚛️ Quantum Computing Basics

This repository contains fundamental experiments in quantum computing using Qiskit.

## Topics Covered

- Superposition
- Quantum Entanglement
- Bell States
- Introduction to Variational Quantum Eigensolver (VQE)

## Technologies

- Python
- Qiskit
- Matplotlib

## How to Run

1. Install dependencies: pip install -r requirements.txt
2. Run a script: python 01_superposition.py

## Objective

This repository documents my learning journey in quantum computing and quantum algorithms.
